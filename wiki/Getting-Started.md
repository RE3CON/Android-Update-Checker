# 🚀 Getting Started

To get started with the **Android Update Checker**, follow these simple steps:

## 1. Access the Dashboard
Open the [Android Update Checker](https://ais-dev-krnbjvxhtxj63l5fteqdnx-75506219897.europe-west2.run.app) in your web browser.

## 2. Connect GitHub (Optional)
If you want to track apps from GitHub repositories or check for beta releases, it's recommended to connect your GitHub account. This helps avoid API rate limits.
*   Click the **"Connect GitHub"** button in the header.
*   Authorize the application.

## 3. Add Your First App
You can add apps in two ways:
*   **Manual Add:** Click the **"Add App"** button and enter the app name, package name, and update URL.
*   **Import:** Click the **"Import"** button to upload an existing list from **App Manager** or **SD Maid SE**.

## 4. Check for Updates
Once your apps are added, click the **"Check All"** button to scan for the latest versions across all sources.
