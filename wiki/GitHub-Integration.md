# 🔗 GitHub Integration

The **Android Update Checker** features a powerful integration with GitHub, allowing you to track apps from repositories and check for beta releases.

## 1. Connect GitHub Account
To use GitHub-related features, you should connect your account. This helps avoid API rate limits and gives you access to more information.
*   Click the **"Connect GitHub"** button in the header.
*   Authorize the application.

## 2. Track GitHub Apps
When you add an app with a GitHub source, the application will automatically check for the latest releases.
*   **Stable Releases:** The app will scan for the latest non-prerelease tag.
*   **Beta Releases:** If you're looking for beta versions, the app will look for prerelease tags.

## 3. GitHub Beta Artifacts
If a repository uses GitHub Actions to build beta artifacts (e.g., APKs), the application can fetch them directly.
*   Enter the **GitHub Owner** and **Repo Name** in the GitHub Connect section.
*   Click **"Resolve"** to find the latest beta artifact.
